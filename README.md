# YouTube + Web Store analytics dashboard

Prototype analytics dashboard combining **YouTube channel metrics** (views,
revenue) and **web store metrics** (units sold, revenue, conversion rate),
with a third **combined** view that overlays both sources.

Built to the Fleetmule design system spec in [design.md](design.md): indigo-violet
primary, soft shadows, `rounded-xl` cards, thin-stroke Lucide icons, sentence-case
copy, class-based dark mode.

## Quick start

```bash
cd app
npm install
npm run dev    # → http://localhost:5173
```

## The three sections

| Page         | Route       | What it shows                                                                                |
| ------------ | ----------- | -------------------------------------------------------------------------------------------- |
| **Overview** | `/`         | Combined revenue (YouTube + store), revenue mix donut, by-source ranking, leaderboards       |
| **YouTube**  | `/youtube`  | Views, ad revenue, RPM, subscribers · per-channel comparison · views ranking · leaderboard   |
| **Web Store**| `/store`    | Revenue, units, conversion, AOV · visitors-to-orders funnel · per-store comparison · leaderboard |

Every page has a **source picker** (single channel/store or roll up "all") and a
**date range picker** (7 / 30 / 90 days, with previous-period delta comparisons).

## Stack

React 19 · TypeScript · Vite 7 · Tailwind CSS v4 · recharts · shadcn-style
primitives · `next-themes` for class-based dark mode · react-router-dom.

## Data

All mock — channel and store names are made up, daily series are generated by
a seeded PRNG in [`app/src/lib/mock-data.ts`](app/src/lib/mock-data.ts) so demos
are stable across reloads. Numbers are scaled to feel plausible for mid-sized
creator businesses (RPM in $2–$8, AOV $38–$102, conversion 1.8%–4.6%).

## Deploy

`app/vercel.json` is already set up with the SPA rewrite. From inside `app/`:

```bash
npx vercel@latest login      # one-time
npx vercel@latest --prod
```

Or import the repo at [vercel.com/new](https://vercel.com/new) with Root
Directory set to `app/` (framework preset = Vite, auto-detected).
